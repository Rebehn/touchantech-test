# touchantech-test
###Built Using
- HTML
- CSS
- Javascript
- MongoDB
- Express
- AngularJS
- Node.js

###To get started:
```
$ git clone git@github.com:Rebehn/firmstep-test.git
$ cd firmstep-test
$ bower i
```
If you are given a choice between angular versions choose version angular#1.6.1.

```
$ npm i
$ nodemon
```
Then open up your browser and navigate to localhost:8000 to view.
